# WP Common

Useful common classes for WordPress development.

## Installation

```bash
composer require humanik/wp-common
```

## Configuration

The library provides a set of common utilities and helpers for WordPress development.

## Development

### Requirements

- PHP 8.3 or higher
- Composer
- Node.js and npm

### Setup

1. Install PHP dependencies:

```bash
composer install
```

2. Install JavaScript dependencies:

```bash
npm install
```

### Running

Start the development environment:

```bash
npm start
```

### Testing

Run PHP tests:

```bash
npm test
```

### Linting

Run PHP linting:

```bash
composer phpcs
```

Fix PHP code style issues:

```bash
composer phpcbf
```

Run static analysis:

```bash
composer phpstan
```
